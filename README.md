# Quasar Balloon Pop

## Tools Used
* [Quasar](http://quasar-framework.org/)
* [Vue.js](http://vuejs.org/)
* [Vuex](http://vuex.vuejs.org/)

## Build Setup

``` bash
# install dependencies
$ npm install

# serve with hot reload at localhost:8080
$ quasar dev

# build for production with minification
$ quasar build

# lint code
$ quasar lint
```
